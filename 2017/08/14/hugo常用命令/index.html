hugo常用命令

14 Aug, 2017 · by author · Read in about 1 min · (23 words)
hugo

编辑

新生成一篇文章

hugo new post/xxx.md

本地调试

hugo server -b localhost --buildDrafts

#发布

1执行hugo 发布命令，需要替换本地域名为外网域名，否则发布的应用，外网点击后访问的是内网地址。

hugo -b http://xxx.xxx.com/123

2 当前文件头部draft=true 修改为 draft=false 3上传public 中生成的静态网页到服务中
